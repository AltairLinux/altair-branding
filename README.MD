# Altair Linux Branding Repository

Professional branding assets and guidelines for **Altair Linux**, a modern educational Linux distribution under Zorvia. This repository centralizes all visual identity elements for consistent use across desktop environments, ISOs, documentation, and marketing materials.

---

## Logo Guidelines

- **Primary Logo:** `logo/altair-logo.svg`
- **Monochrome Logo:** `logo/altair-logo-monochrome.svg`
- **Icon Version:** `logo/altair-logo-icon.svg`
- **Raster Fallback:** `logo/altair-logo.png`

**Usage Rules:**
- Maintain clear space around the logo.
- Do not distort, rotate, or recolor outside the approved palette.
- Use monochrome logo for single-color contexts or print.

---

## Color Palette

| Role       | Hex       | RGB          | Usage                         |
|------------|-----------|--------------|-------------------------------|
| Primary    | #0B1D3A   | 11,29,58     | Main logo and headers         |
| Accent     | #3FD0FF   | 63,208,255   | Highlights, stars, links      |
| Secondary  | #FFC857   | 255,200,87   | Buttons, secondary accents    |
| Neutral    | #F5F7FA   | 245,247,250  | Background, light UI elements |

**Usage Guidelines:**
- Primary color dominates branding.  
- Accent used sparingly for emphasis.  
- Neutral supports light backgrounds.  

---

## Typography

- **Headings:** Open Sans Bold (or system sans-serif)
- **Body:** Open Sans Regular
- Maintain consistent hierarchy:
  - H1: 48px
  - H2: 36px
  - H3: 24px
  - Body: 16px
- Ensure high contrast between text and background.

---

## Wallpapers

- **Dark Mode:** `wallpapers/desktop-dark.png` — monochrome cosmic scene, central bright cyan star.  
- **Light Mode:** `wallpapers/desktop-light.png` — pastel sky, soft gradient, central cyan star.  

**Notes:** Use wallpapers as default desktop backgrounds. Maintain contrast for icons and readability.

---

## Icons & Mascots

- **Icons:** `icons/` folder contains cursor, system icons, and misc placeholders.  
- **Mascots:** `mascots/` folder reserved for optional vector mascots.  

**Guidelines:**
- Minimal, flat style consistent with modern Linux aesthetics.
- Stroke weight and color palette must be consistent.

---

## Contribution Guidelines

- Add new assets in correct folders with descriptive filenames.
- Update `docs/logo-guidelines.md` for any logo variations.
- Keep color palette consistent; submit pull requests for changes.
- Review existing files for compliance with Altair branding standards.
