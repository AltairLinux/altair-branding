# Altair Linux Color Palette Usage Guide

This guide defines the approved color palette for **Altair Linux** branding. Use consistently across logos, wallpapers, icons, and UI elements.

---

## Color Roles

### Primary
- **Color:** Deep Space Navy (#0B1D3A)
- **Usage:** Main logo lines, headers, primary backgrounds
- **Notes:** Dominant color, high contrast, maintain clarity

### Accent
- **Color:** Stellar Cyan (#3FD0FF)
- **Usage:** Highlights, stars, interactive elements
- **Notes:** Use sparingly for emphasis

### Secondary
- **Color:** Solar Gold (#FFC857)
- **Usage:** Buttons, secondary accents, minor highlights
- **Notes:** Optional, subtle emphasis only

### Neutral
- **Color:** Soft White (#F5F7FA)
- **Usage:** Text, light backgrounds, UI panels
- **Notes:** Ensure sufficient contrast for readability

### Secondary Neutral
- **Color:** Light Gray (#B0B8C2)
- **Usage:** Secondary text, disabled elements
- **Notes:** Minimal contrast for background layering

---

## Guidelines

1. **Consistency:** Apply the palette consistently across all branding materials.  
2. **Accessibility:** Ensure text and UI elements meet readability standards.  
3. **No unauthorized colors:** Do not introduce colors outside this palette in logos or UI.  
4. **Flat and minimal:** Avoid gradients for logos; only use gradients if specified in wallpapers.
